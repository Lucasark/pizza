TELA PRINCIPAL: 

	INFORMAR SHOWS COM BILHETES À VENDA

	SELECIONOU O SHOW 

	NA TELA APARECERÁ BANDA, TURNÊ, DATA/LOCAL

	SELECIONOU DATA/LOCAL

	APARECERÃO OS TIPOS DE INGRESSO NOS LOCAIS

	SELECIONOU O TIPO 

	INFORMARÁ QUANTOS BILHETES IRÁ QUERER

	PODERÁ REPETIR TUDO ISSO OU FINALIZAR A COMPRA

	DEPOIS DIGITAR CARTÃO DE CRÉDITO COM 4 BLOCOS E 4 DIGITOS + VALIDADE(MM/YY) + CODIGO DE SEGURANÇA

	

	EXIGÊNCIAS: 
		3 SHOWS EM CADA UM EM 3 CIDADES COM 3 DATAS.

		1 CIDADE SÓ POSSUI UM ESTÁDIO

		ESTÁDIOS POSSUEM INGRESSOS DE TIPOS DIFERENTES

		CARRINHO COM DOIS INGRESSOS DIFERENTES E QUANTIDADES VARIADAS

		ENTIDADE VALIDADORA DE CARTÃO 

		INTERFACE EM SYSTEM.OUT.PUT E JAVA.UTIL.SCANNER


CLASSES: 


	CLASSE CONCERTO ( CONTERÁ CLASSE BANDA/TURNÊ  )
	
	CLASSE EVENTO ( CONTERÁ CLASSE DATA/LOCAL ) 

        CLASSE ASSENTO ( CONTERÁ TIPOS DE LUGARES NO ESTÁDIO )

	CLASSE COMPRA ( CONTERÁ ESCOLHA DE QUANTIDADE ENVIANDO O PREÇO)

        CLASSE CARRINHO ( ARMAZENARÁ UM ARRAY QUE TERÁ DADOS DE: 

	BANDA/TURNÊ, DATA/LOCAL, TIPO DE LUGAR, QUANTIDADE, TOTAL )

	CLASSE CONFIRMAR ( DADOS DO CARTÃO )

        FIM!
